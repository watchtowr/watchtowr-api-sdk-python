# ClientBusinessUnitRuleDetailKeywordRuleType

Keyword rule type. HOSTNAME: matches domain/subdomain names (default). CNAME: matches CNAME DNS record values. TLS_SSL: matches TLS/SSL certificate subject names.


## Values

| Name       | Value      |
| ---------- | ---------- |
| `HOSTNAME` | HOSTNAME   |
| `CNAME`    | CNAME      |
| `TLS_SSL`  | TLS_SSL    |
